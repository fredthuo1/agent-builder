# build-a-task-app-with-au

Generated by **Autonomous Multi-Agent App Builder**.

## Features
- Auth (demo token)
- Tasks CRUD
- SQLite storage
- Responsive UI

## API
- POST /api/login — Get demo token
- GET /api/tasks — List tasks
- POST /api/tasks — Create task
- DELETE /api/tasks/:id — Delete task

## Run
### Backend
```bash
cd backend
npm i
npm run dev
```

### Frontend
```bash
cd frontend
npm i
npm run dev
```
